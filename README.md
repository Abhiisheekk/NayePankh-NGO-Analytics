# 📊 NayePankh Foundation NGO Analytics Dashboard

## 📌 Project Overview

The **NayePankh Foundation NGO Analytics Dashboard** is an end-to-end Business Intelligence project developed using **SQL Server** and **Power BI**. The project analyzes NGO operations, fundraising performance, program effectiveness, geographic outreach, and overall social impact through interactive dashboards.

This solution demonstrates how data analytics can help NGOs monitor key performance indicators, improve resource allocation, and support data-driven decision-making.

---

## 🎯 Objectives

* Analyze donation trends and fundraising performance.
* Monitor beneficiary outreach across various programs.
* Evaluate volunteer participation and program effectiveness.
* Compare geographic performance across multiple cities.
* Measure the social impact created by the NGO.

---

## 🛠️ Tech Stack

* **SQL Server**
* **Power BI**
* **Microsoft Excel**
* **DAX (Data Analysis Expressions)**

# 📊 Dashboard Pages

Provides an overview of the NGO's overall performance through KPIs and trend analysis.

* Total Volunteers

* Average Satisfaction

### Visualizations

* Monthly Donation Trend
* Program-wise Beneficiaries
* Interactive Filters

# 📊 Key Insights

* Individual donors contributed the largest share of donations.
* Food Distribution and Education programs reached the highest number of beneficiaries.
* Donations showed seasonal growth toward the end of the year.
* Major cities generated the highest fundraising contributions.
* The NGO created significant social impact through food distribution, clothing drives, sanitary kit distribution, and animal welfare initiatives.

1. Clone this repository.
2. Open **NayePankh_Analytics.pbix** in **Power BI Desktop**.
3. Open the SQL scripts in SQL Server Management Studio (SSMS) to recreate the database if required.
4. Refresh the Power BI dataset.
